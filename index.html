Keepr

Vue Frontend / C# Backend

Source Code

Test Account - Username: zak1@test.com | Password: Clarity94!

Keepr, much like Pinterest, offers its users a platform to gather and preserve ideas and inspiration by curating a collection of "keeps" in personalized vaults. With Keepr, users can easily explore new concepts, save their favorite items, and organize their discoveries in a visually appealing way, providing them with a rich source of creativity and motivation. Utilized Vue.js frontend with Vue-Router to manage the DOM. On the server side, I used the DotNet WebApi, implementing the Repository pattern to communicate with the database. Other technologies used were Auth0, Dapper as an ORM, and a MySQL server for my database.

GamersOnly

Vue Frontend / Node.js Backend

Source Code

Test Account - Username: zak1@test.com | Password: Clarity94!

Gamers Only is a full stack web application that I made with my classmates at Boise Codeworks as part of our capstone project. We utilized partner programming as well as Scrum, had a Trello board tracking development progress and distributing tickets, and created a high fidelity Figma mock up to base our design off of. The website allows you to create an account through Auth0, search through an almost infinite number of games that are pulled from the RAWG API. A user can create a group for a game that other users can join, and can also create events within that group for the players to participate in. The group members can also participate in a live group chat amongst themselves.
